# 《赤心巡天》关键剧情路线图（第1-500章）

> 本文档整理自小说原文，按照主线剧情发展脉络梳理关键事件与人物。
> 证据出处主要来自：`赤心巡天(1-500章).txt`，第498-500章补自 `赤心巡天(501-1000章).txt`

---

## 一、剧情总览

1-500章主要讲述主角姜望从庄国枫林城的垂死乞丐，一步步踏上修行之路，经历庄国白骨道之乱后，离开故土前往齐国，并参与七星楼秘境，先后经历森海源界、隐元星沙海与天枢星浮陆生死棋开局的故事。

### 核心线索
1. **个人成长线**：姜望的修行突破与心性成长
2. **白骨道灭门线**：贯穿庄国篇的核心冲突
3. **太虚幻境线**：姜望在虚拟世界的冒险与成长
4. **七星楼秘境线**：北斗星位开启，多世界探索与资源争夺
5. **亲情线**：姜望与妹妹姜安安的羁绊

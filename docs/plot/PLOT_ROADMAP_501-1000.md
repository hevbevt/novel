## 一、剧情总览

501-1000章承接七星楼秘境终局，姜望回归临淄后卷入名门商战与刺杀风波，在东街口与王夷吾正面交锋并叩开内府，摘得神通“三昧真火”，确立“天下第一腾龙”的名声。随后公开约战雷占乾收尾，又在青牌身份掩护下离开临淄，前往南遥与青羊镇，卷入北衙追缉与尹观出境交易，边境遭捕神岳冷拦截。地狱无门阎罗分兵引发静海高、天府城血案，盲眼打更人出手连斩阎罗；并通过松林兽巢揭示开脉丹体系与妖族/凶兽关系。其后近海群岛风波牵动海宗明追杀，姜望驼峰山伏杀外楼巅峰；返云国与安安团聚并开脉，迟云山传承争夺开启、云顶仙宫残骸入手，庄国旧怨与望江城暗线重新浮出。继而祁昌山脉矛盾转为庄雍国战，姜望除夕夜潜入新安斩董阿，锁龙关大战韩殷战死、庄国夺永昌郡；韩煦得墨门扶持立殷歌城，祝唯我叛国，杜如晦追凶。姜望心魔暴走昏迷后潜入清江水府，发现水底魔窟与宋横江疑云。随后水府魔窟冲突升级，庄承乾与白骨尊神的无生劫对弈摊开，姜望自毁神魂设局反杀并叩开二府；宋横江身死，宋清约继位，庄高羡追杀受阻。姜望返云国安顿安安，云霄阁归位推动云顶仙宫复苏，并转向雍国青云亭。其后姜望潜入雍国顺安府，威宁候寿宴与青云亭线交织，人魔屠灭青云亭、平衡之血线索浮出，姜望夺回青云亭遗楼并得仙术“平步青云”。他经和国返临淄，重玄胜开内府神通“法天象地”，海族/迷界与钓海楼海祭线全面铺开，姜无邪条件未合，姜无忧允诺出手。天涯台海祭翻案后危寻现身，镇海盟成立；姜望与碧珠婆婆被投入迷界丁未洗罪，姜望反杀碧珠婆婆并猎杀统帅级海族，白象王率军围岛索人，丁未浮岛大战将起。随后姜望突围并与褚密分离，褚密舍身搭星桥送其过界河，姜望斩杀水鹰嵘、鱼嗣庆等完成洗罪；迷界位移将其带至“浮图净土”，得陈艮、阎伽相助后转道旸谷，返回丁未乘灼日飞舟出迷界，途中遭血王截杀被武道强者王骜拦下，危寻等人以姜望为饵引“万瞳”的布局显露失败。返怀岛后竹碧琼油尽灯枯，姜望开第三内府摘得神通“不周风”，送其葬入天府秘境并立愿。海勋榜推出，姜望列副榜第一；他试剑钓海楼，先败包嵩，后与季少卿订生死决斗。天涯台决战中姜望破上弦月、破天门、以神魂匿蛇截断季少卿筑楼，最终废其修为并坐守天涯台，誓要“熬死”季少卿。天涯台余波中姜望熬死季少卿并离岛清债；五仙门被屠引出地狱无门与田家暗线，范清清投奔并献万仙宫残卷；竹碧琼自天府秘境复活，镜花水月疑云与竹素瑶线浮出，最终回钓海楼拜辜怀信为师。重玄家族内斗升温，太虚派虚泽甫来访授太虚使者身份与太虚角楼图纸，齐国公开角楼落地天府城，重玄胜操盘商业配套。姜望冲击太虚论剑台排名，与左光殊河伯神通交锋，并开始谋求黄河之会名额。（本文档按每50章更新，当前覆盖501-1000章）

### 核心线索
1. **修行突破线**：腾龙圆满→内府境突破→神通成形
2. **临淄权斗线**：重玄胜与重玄遵势力此消彼长，商战与舆论攻防
3. **离境追缉线**：青牌身份掩护、北衙追缉升级、边境阻截
4. **地狱无门线**：尹观布局与阎罗分兵内耗
5. **帝室观战线**：皇子皇女观战与态度释放
6. **开脉丹体系线**：妖族、凶兽与开脉丹来源被揭示
7. **迟云山传承线**：神通果与云顶仙宫秘闻启动
8. **云顶仙宫复苏线**：灵空殿/青云亭/凌霄阁三宗遗留成为关键
9. **庄雍国战线**：由边境矛盾引爆的国战与版图重划
10. **师徒因果线**：董阿与姜望的师徒决裂与复仇
11. **心魔与姜魇线**：心魔成形、姜魇介入与生死共担
12. **清江水府魔窟线**：水底阴魔与宋横江疑云
13. **无生劫对弈线**：庄承乾与白骨尊神的生死博弈
14. **青云亭覆灭与人魔线**：平衡之血、人魔与卦师线索
15. **海祭与海族线**：竹碧琼海祭、迷界与沧海旧史揭示
16. **太虚幻境扩张线**：太虚派来访、太虚使者与太虚角楼体系启动
17. **镜花水月复生线**：竹碧琼复活、天府秘境泡影与竹素瑶线索
18. **田家暗线**：乌列追查田焕文，田氏与近海格局交织

第501-750章完成七星楼余波收束与临淄声名确立：姜望连败王夷吾、雷占乾并摘三昧真火，帝室与名门关注升温；青牌离境与尹观出境交易牵动追缉风暴，阎罗分兵与打更人出手引发静海高/天府城惨案，松林兽巢揭开开脉丹体系隐秘。随后海宗明追杀引出驼峰山伏杀，姜望借定风珠与三昧真火斩外楼巅峰；返云国与安安团聚并开脉，迟云山传承争夺开启，云顶仙宫残骸与复苏线索落入姜望掌中。祁昌山脉矛盾升级为庄雍国战，姜望除夕夜弑师董阿，迫使战局提前决战，锁龙关破、韩殷战死、庄国夺永昌郡；韩煦借墨门立殷歌城，祝唯我叛国。随后清江水府魔窟冲突升级，庄承乾与白骨尊神无生劫对弈摊开，姜望自毁神魂设局反杀并叩开二府，宋横江身死、宋清约继位；庄高羡追杀姜望被苦觉拦阻。姜望返云国安顿安安，云霄阁归位推动云顶仙宫复苏，并转向雍国青云亭线索。

第751-800章进入雍国青云亭线：姜望在顺安府遭遇神秘人，获知无生教与王长吉线索；威宁候寿宴权斗升级，青云亭被九大人魔屠门，封池两脉覆灭，姜望夺回青云亭遗楼并获仙术“平步青云”，救出封鸣。人魔“平衡之血”与卦师“仙主”线索浮出，云顶仙宫三殿齐动。姜望经和国返临淄，重玄胜神通“法天象地”出世；海族/迷界与钓海楼海祭线铺开，姜无邪条件未合，姜无忧承诺全力相助，姜望回北衙复命并筹备出海。

第801-850章转入近海任务与海祭前置：姜望接“金针案”被林有邪并案同行，念尘锁定武一愈，尹观咒杀灭口，地狱无门近海暗线浮出。其后行至有夏岛遭遇海兽失控与五仙门/怒鲸帮对峙，借正声殿潮音自创并刻印甲等上品道术“八音焚海”；小月牙岛揭破“难说大师”骗局，钓海楼大师兄陈治涛出面善后并提及长老层级。姜无忧抵近海训晏抚、表态参与海祭，姜望随碧珠婆婆入囚海狱探监竹碧琼，海祭营救继续推进。

第851-900章进入海祭翻案与迷界洗罪：天涯台海祭上姜望为竹碧琼翻案，碧珠婆婆被定罪贬入迷界；钓海楼主危寻现身裁决，判竹碧琼同罪并逼姜望替她洗罪，两人同入丁未区域。姜望获指舆与诸多资源后在迷界适应异化规则，设局击杀碧珠婆婆，得行思龙头杖、云暮樽与五色鱼。浮岛得知迷晶与海疆榜机制，并遭鱼万谷统帅级海族围猎，借八音焚海与许象乾“锦绣”救命反杀，杀六逼逃三。丁景山关注海疆榜、公开力挺不交人，白象王率海族大军围岛索人，丁未区域“敌袭”爆发，海战一触即发。

第901-950章承接丁未浮岛大战：姜望突围与褚密分离，褚密舍身搭星桥，姜望于界河前斩杀水鹰嵘、鱼嗣庆并完成洗罪；迷界位移使其进入浮图净土，得陈艮、阎伽协助后返丁未，乘灼日飞舟出迷界，遭血王截杀被王骜拦下。回怀岛后竹碧琼油尽灯枯，姜望开第三内府摘不周风，将其葬入天府秘境并立愿。海勋榜推出，姜望列副榜第一；他试剑钓海楼，败包嵩，与季少卿签生死约战天涯台，破上弦月与天门、断其筑楼，覆军与沉都对峙之下废其修为并坐守天涯台“熬死”。

第951-1000章进入天涯台余波与太虚角楼启幕：姜望熬死季少卿后离岛清债，五仙门被屠，范清清投奔并献万仙宫残卷；乌列、林有邪出海线牵出田焕文与田家暗线。竹碧琼自天府秘境复活，镜花水月与竹素瑶线浮出，最终回钓海楼拜辜怀信为师并显天门。重玄家族内斗升温，重玄元祜露面；太虚派虚泽甫来访授太虚使者与太虚角楼图纸，齐境公开角楼落地天府城，吕宗骁与重玄胜操盘商业配套。姜望解放神通冲击太虚论剑台排名，与左光殊河伯神通交锋，并开始谋求黄河之会名额。

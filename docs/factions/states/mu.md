# 牧国 (Mu)

- 势力类型：国家/朝廷
- 活跃范围：北域霸主国之一
- 核心机构：神殿体系与军政合一结构
- 主要力量：金冕祭司与牧国军伍天骄

## 文本明确
- 牧国更换内府出战者为“邓旗”，其在观河台一路晋级四强。
- 牧国在三十岁以下无限制场宣布弃赛，以回应景国夺魁布局。

## 合理推断
- 牧国试图以“更换名额”与神殿体系稳住观河台话语权。

## 关键事件（1101-1200章）
- 邓旗（赵汝成）代表牧国出战内府正赛，战至四强。
- 牧国在太虞真人现身后宣布无限制场弃赛。

## 与主角关系
- 邓旗与姜望同为内府正赛核心天骄，且牵动枫林旧线。

## 现状（至第1200章）
- 牧国在内府场有亮眼表现，但无限制场选择避让。

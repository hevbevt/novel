# 庄国 (Zhuang)

- 势力类型：国家/朝廷
- 活跃范围：西境道属国，清河郡为旧土核心
- 核心机构：庄国朝廷、国相体系、道院体系
- 主要力量：庄高羡、杜如晦、董阿（至其身死）、国院与边军

## 文本明确
- 庄国体量不大但军势强，长期与雍国在锁龙关对峙。
- 庄雍国战中庄高羡御驾亲征，锁龙关决战斩韩殷，庄国夺永昌郡。
- 国战后杜如晦追查董阿凶手，清江水府魔窟事件牵动庄国高层。
- 段离以“叛国”名义被处决，杜野虎被保为九江玄甲核心。

## 合理推断
- 庄廷内部对枫林旧案的处理存在长期政治禁区，导致对枫林城域出身者的系统性疑虑。

## 关键事件（1-750章）
- 白骨道之乱引发枫林城域覆灭。
- 庄雍国战爆发并改写西境版图。
- 庄廷追杀姜望与清江水府搜查行动同时推进。

## 关键事件（1051-1100章）
- 段离诱引杜野虎立誓，借“叛国案”自赴死局，庄廷对外定性为旧伤复发。
- 杜野虎被塑为忠义虎臣，九江玄甲权力重整。
- 庄国确定林正仁为内府场出战者赴观河台。

## 关键事件（1101-1150章）
- 林正仁在内府选拔赛击败盛国江离梦，取得正赛资格。
- 内府正赛首轮林正仁“血鬼反噬”弃赛，庄国颜面受损，景帝出声不满。

## 与主角关系
- 姜望出身庄国枫林城域，与庄廷因白骨道旧案、弑师董阿而彻底对立。

## 现状（至第1150章）
- 黄河之会内府正赛失利，庄国对外声望受挫；杜如晦带林正仁离场善后。

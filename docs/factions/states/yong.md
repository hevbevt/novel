# 雍国 (Yong)

- 势力类型：国家/朝廷
- 活跃范围：西境大国，旧称强盛
- 核心机构：雍庭、边军体系
- 主要力量：韩殷、韩煦、雍国诸军

## 文本明确
- 雍国曾为强国，庄雍国战中与庄国激烈对峙。
- 锁龙关决战韩殷战死，雍国被迫割让岭北并转入守势。
- 韩煦掌权后引入墨门，殷歌城建立并立墨学为国学。
- 顺安府为雍国西部边府，战后改革推进，儒道寺体系被拆改，墨门影响力受控。
- 威宁候焦武寿宴上与武功侯薛明义交锋，雍庭借礁国奸细话题敲打藩镇。
- 雍国随后对礁国用兵，焦武领军成为前线主帅之一。

## 合理推断
- 雍国内部权力结构由宗室独断转向“君权+墨门”共治雏形。
- 新旧势力（武功侯系与威宁候系）仍在拉扯，墨门态度成为关键变量。

## 关键事件（1-800章）
- 庄雍国战爆发，锁龙关决战与岭北割让。
- 韩煦切断国力、改制革政，墨门深度介入。
- 威宁候寿宴权斗与礁国奸细事件。
- 对礁国用兵，西境战线再起。

## 与主角关系
- 姜望计划前往雍国获取云顶仙宫遗留“青云亭”。
- 姜望在顺安府活动，借威宁候寿宴切入青云亭。

## 现状（至第800章）
- 韩煦实控雍政，改革与权斗并行，对礁国用兵。

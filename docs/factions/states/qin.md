# 秦国 (Qin)

- 势力类型：国家/朝廷
- 活跃范围：西域强国，咸阳为都
- 核心机构：皇室、军府与镇狱体系
- 主要力量：秦帝与秦国皇族、军伍天骄

## 文本明确
- 秦国派秦至臻参与黄河之会内府场正赛。
- 秦帝在观河台点名“嬴子玉”，秦国皇室身份线公开。

## 合理推断
- 秦国在黄河之会同时维持天骄战力与皇室血脉布局。

## 关键事件（1001-1200章）
- 秦至臻观河台正赛连胜，半决赛败于姜望。
- 秦帝在观河台点名嬴子玉，引发各国震动。

## 与主角关系
- 秦至臻为姜望内府正赛对手之一。

## 现状（至第1200章）
- 秦国天骄在观河台受挫，但皇室身份线浮出。

# 云国 (Yun)

- 势力类型：国家/朝廷
- 活跃范围：西境中立国，云城与抱雪山
- 核心机构：凌霄阁（宗门即国本）
- 主要力量：叶凌霄、叶青雨与凌霄阁体系

## 文本明确
- 云国以宗门为国本，凌霄阁既是宗门也是国家权力中心。
- 云国对外保持中立，拒绝卷入庄国追凶与国战余波。
- 姜安安被安置在云国，成为凌霄阁亲传弟子。

## 合理推断
- 云国的中立政策以“宗门利益优先”为底层逻辑，优先维护宗门传承与国境安全。

## 关键事件（1-750章）
- 姜望多次进入凌霄秘地，云国成为其避险与补给的重要据点。
- 叶凌霄将云顶仙宫遗楼“云霄阁”交予姜望，间接推动仙宫复苏。

## 与主角关系
- 云国是姜望与姜安安的避风港，但凌霄阁明确划出“不牵扯宗门”的边界。

## 现状（至第750章）
- 云国中立立场稳固，凌霄阁继续坐镇。

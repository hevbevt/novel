# 龙门书院 (Longmen Shuyuan)

- 势力类型：学派/书院
- 活跃范围：宋国与魏国之间，观河台以南
- 核心人物：殷文华、照无颜、子舒（同门）
- 组织定位：儒门书院，天下四大书院之一

## 文本明确
- 观河台落成时，儒门先贤起草祭文，身后弟子在观河台南面结庐讲学，成龙门书院前身。
- 书院西为宋国、东为魏国，北临观河台，与两国距离极近。
- 四大书院与法家三刑宫一样，不禁弟子为官；宋、魏都有大量书院出身的官员。
- 宋国独尊儒术、以礼治国，受龙门书院影响尤深；魏国更重兵家但儒家势力亦强。

## 合理推断
- 龙门书院在黄河之会期间对列国儒修具有聚合与声望放大效应。

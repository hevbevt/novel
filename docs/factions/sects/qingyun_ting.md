# 青云亭 (Qingyun Ting)

- 势力类型：宗门
- 活跃范围：雍国顺安府文溪县一带
- 核心人物：池定方、封越、池陆、封鸣
- 组织定位：云顶仙宫遗楼的守承宗门

## 文本明确
- 青云亭为雍国顺安府宗门，封、池两脉并存。
- 青云亭遗楼与云顶仙宫相关，被制为“钥匙”样式。
- 九大人魔为“平衡之血”屠灭青云亭，宗门覆灭。

## 合理推断
- 封池两脉长期平衡与云顶仙宫遗留机制有关。

## 关键事件（1-800章）
- 威宁候寿宴受辱与封越被扣，青云亭权势受压。
- 人魔突袭，池定方献祭“殃祸乌云”失败，宗门覆灭。
- 青云亭遗楼归位云顶仙宫，姜望获仙术“平步青云”。

## 与主角关系
- 姜望潜入青云亭夺回遗楼，是云顶仙宫复苏关键。

## 现状（至第800章）
- 宗门覆灭，核心传承随遗楼并入云顶仙宫。

# 凌霄阁 (Lingxiao Ge)

- 势力类型：宗门/圣地
- 活跃范围：云国抱雪山、云城
- 核心人物：叶凌霄、叶青雨
- 组织定位：云国国本宗门，维持中立

## 文本明确
- 凌霄阁为云国顶级宗门，与云国政权高度绑定。
- 叶凌霄在庄国追凶线中坚持中立，拒绝直接卷入国战与追杀。
- 凌霄阁拥有云顶仙宫遗留“云霄阁”，并交由姜望。

## 合理推断
- 凌霄阁对姜望的支持以“安安”与“仙宫遗留利益”双重驱动。

## 关键事件（1-750章）
- 叶青雨早期在玉衡峰被姜望所救，云中令成为双方信物。
- 姜安安被收为亲传弟子，成为凌霄阁与姜望的纽带。
- 云霄阁归位，云顶仙宫复苏线推进。

## 与主角关系
- 既是庇护者也是边界制定者，支持姜望但不与庄国正面冲突。

## 现状（至第750章）
- 宗门稳定，中立政策持续；云霄阁已交付姜望。

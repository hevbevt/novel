# 钓海楼 (Diaohailou)

- 势力类型：宗门/海上霸主
- 活跃范围：近海群岛与海域
- 核心人物：危寻、崇光真人、海京平、陈治涛、海宗明、碧珠婆婆、竹碧琼
- 组织定位：近海群岛主导势力，掌控海祭与海域秩序

## 文本明确
- 钓海楼为近海群岛最强势力之一，地位超然。
- 楼主危寻为衍道真君，亲临海祭并裁断争端。
- 楼内有二十四位实权长老，海宗明为其一。
- 实权长老分为“四上八中十二下”，等阶分明。
- 实权长老另分靖海长老、护宗长老、实务长老三类职务。
- 大师兄陈治涛为神临强者，代表宗门处置海上风波。
- 囚海狱为钓海楼重狱，狱卒亦为囚徒。
- 钓海楼定四月四日于天涯台祭海，竹碧琼被列为祭品。
- 天涯台为钓海楼最神圣之地，相传为祖师钓龙之处。
- 海祭为固定传统，每年行祭，祭品多为“无赦之罪”之人。
- 天涯台海祭翻案后，碧珠婆婆被定罪贬入迷界。
- 天涯台决斗后季少卿被姜望熬杀，辜怀信派系受挫。
- 竹碧琼自天府秘境复活后回归钓海楼，拜辜怀信为师。

## 位阶与内部结构
### 文本明确
- 二十四位实权长老：宗门核心权力层（海宗明为其一）。
- 长老体系：碧珠婆婆为长老之一，掌门下弟子竹碧琼。
- 长老分级：四上八中十二下（层级化治理）。
- 大师兄陈治涛：神临强者，宗门年轻一代的门面人物。
### 合理推断
- 宗门以海祭与长老体系为权力枢纽，内部层级分明。

## 合理推断
- 钓海楼对外极强势，近海群岛诸宗多需避其锋芒。

## 关键事件（1-900章）
- 海宗明追杀姜望，被驼峰山伏杀。
- 竹碧琼因泄露海宗明行踪被废修为、逐出门墙并定为海祭祭品。
- 海祭线引发姜望与齐国帝室的资源调动。
- “难说大师”骗局暴露后，陈治涛出面善后并承诺补偿受骗者。
- 天涯台海祭翻案，危寻现身裁决，碧珠婆婆被贬迷界。

## 与主角关系
- 海宗明与姜望结怨，姜望为救竹碧琼筹备对抗钓海楼海祭。

## 现状（至第1000章）
- 危寻主导成立镇海盟，钓海楼影响力进一步外扩。
- 海祭争端落定，碧珠婆婆被贬迷界后身死，陈治涛声望上升。
- 竹碧琼复归并入辜怀信门下，钓海楼内部派系格局生变。

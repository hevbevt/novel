# 平等国 (Pingdengguo)

- 势力类型：隐秘组织/邪教
- 活跃范围：齐国及周边；曾与夏国联络
- 核心人物：崔杼、张咏、神秘强者（星楼操控者）
- 组织定位：以“平等/公正”理念动员人心，暗中破坏齐国秩序

## 文本明确
- 崔杼刺君与张咏哭祠后，平等国浮出水面并被齐庭定性为邪教。
- 组织曾联络夏国，齐帝发动对夏战争后，夏国交出其神临成员以自证清白。
- 借星光圣楼投射力量影响姜望道心（星月原）。

## 合理推断
- 具备跨域渗透与“理念蛊惑”能力。

## 关键事件（1051-1100章）
- 齐夏战局收束，平等国神临成员被夏国交出，组织浮出水面。
- 星月原以星楼方式影响姜望，被观衍破局；内部密谈调整策略，转向“利用”姜望。

## 与主角关系
- 试图以理念蛊惑姜望，为组织塑造“新崔杼”候选；与姜望形成对立。

## 现状（至第1100章）
- 按内部指令暂避风头，短期内避免直接刺杀姜望，策略偏向渗透利用。

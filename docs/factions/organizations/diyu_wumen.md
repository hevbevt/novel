# 地狱无门 (Diyu Wumen)

- 势力类型：组织/杀手与情报集团
- 活跃范围：跨国活动，齐国与边境线为重点
- 核心人物：尹观（秦广王），宋帝王、平等王、卞城王、都市王、仵官王等
- 组织定位：以“阎罗”体系运作的地下势力

## 文本明确
- 地狱无门内部存在离心与派系对立，阎罗分兵导致内耗。
- 组织在齐国边境线被北衙追缉，损失惨重。
- 仵官王与“小鬼”在近海群岛活动，尹观可远程施展咒术灭口。

## 合理推断
- 组织内部权威受损，后续可能出现新的整合或分裂。

## 关键事件（1-850章）
- 阎罗分兵引发静海高、天府城血案；打更人出手连斩阎罗。
- 尹观出境交易与姜望短期合作。
- 近海群岛武一愈被擒后遭尹观咒杀，地狱无门近海暗线暴露一角。

## 与主角关系
- 姜望在尹观出境交易中充当协助者，但非长期同盟。

## 现状（至第850章）
- 组织受到重创但仍在近海保有隐秘活动能力。

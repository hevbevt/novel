# 北衙 (Beiya)

- 势力类型：官方执法/追缉体系
- 活跃范围：齐国边境与要冲
- 核心人物：郑世、岳冷、厉有疚
- 组织定位：皇朝隐秘执法力量，对超凡犯罪进行打击

## 文本明确
- 北衙在边境线拦截尹观，岳冷与尹观正面对决。
- 北衙都尉郑世奉旨彻查崔杼刺君案，获授权调用“打更人”。
- 捕神岳冷携镇府重器“天罗”随行，星月原护持姜望并封锁现场。
- 朱禾之盟赋予齐国神临级青牌涉案时跨境追缉权限，后推及东域。

## 合理推断
- 北衙具有跨区域行动权限，能够牵动地方秩序与商路安全。

## 关键事件（1-1100章）
- 尹观出境交易被北衙拦截，冲突升级。
- 刺君案后北衙问话姜望并接管太庙线索。
- 星月原护持行动中，岳冷与厉有疚暗中随行，追缉平等国未果。

## 与主角关系
- 姜望协助尹观出境过程中与北衙正面冲突。

## 现状（至第1100章）
- 郑世继续主导刺君案收尾与平等国线调查，青牌体系影响力进一步上升。

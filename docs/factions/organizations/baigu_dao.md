# 白骨道 (Baigu Dao)

- 势力类型：邪教/隐秘组织
- 活跃范围：庄国与西境为主要舞台
- 核心人物：白骨尊神、白骨道子（王长吉）、白骨使者（张临川）、白骨圣女（妙玉）
- 组织定位：以幽冥神祇降世为目标的邪教体系

## 文本明确
- 白骨道以冥烛与白骨道子为核心，长期谋划现世降临。
- 枫林城域覆灭与白骨道密切相关，成为庄国篇主线冲突。
- 庄承乾与白骨尊神的无生劫对弈，是白骨道体系的延伸与深化。
- 雍国线出现“无生教”传闻，疑由张临川另立门户。

## 合理推断
- 白骨道在现世遭受重创后转向更隐秘的布局，仍保有长期渗透能力。
- 无生教可能是白骨道残余势力的再整合形态。

## 关键事件（1-800章）
- 枫林城域覆灭与白骨道子成形。
- 清江水府魔窟中白骨尊神短暂降临，引爆无生劫。
- 无生教在雍国活动线索出现，张临川改貌传闻浮出。

## 与主角关系
- 姜望早期与白骨道直接对抗，后期被卷入无生劫对弈。

## 现状（至第800章）
- 白骨尊神退回幽冥并孕育道胎，白骨道残余势力疑以无生教名义活动。

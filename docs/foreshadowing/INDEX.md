# 伏笔库总览

本目录用于统一管理《赤心巡天》伏笔线索与回收进展。

## 使用说明
- 伏笔条目按卷维护，详见各 `TRACKER_*.md`。
- 主题合并总表用于跨卷合并同一伏笔线索。
- 剧情路线图仅保留简短索引，不再重复维护伏笔正文。

## 状态说明
- 追踪中：尚未回收或未完成闭环。
- 推进中：已有明显进展但未结论。
- 已回收：明确收束或线索闭环。
- 反转：线索被证伪或方向改变。
- 作废：后续不再延续。

# 姜望 (Jiang Wang)

- 叙事重要度：主角
- 初登场：第1章
- 身份/出身：庄国清河郡枫林城凤溪镇出身，家境清贫，有妹妹姜安安
- 所属/阵营：早期庄国道院；后入齐国，受封“青羊男”
- 修行/能力：四府神通内府；主修剑道与道术；神通为三昧真火、歧途、不周风；内府秘藏“追风”；擅人道之剑、朽木决；获仙术“平步青云”；自创甲等上品道术“八音焚海”，悟出并刻印“声闻仙态”；修成“火界”与瞳术“干阳之瞳”，内府决赛中现“剑仙人”形态并合“人字剑”；太虚幻境获“最强内府”荣名；掌行思龙头杖与云暮樽并驭五色鱼
- 关键经历：拾得左光烈遗物服开脉丹；继承太虚洞真墟福地；枫林城域陷落后远行齐国并迅速崛起；临淄生死战斩王夷吾并开内府；迟云山争夺得云顶仙宫残骸；除夕夜新安城斩杀董阿；清江水府魔窟反杀庄承乾并叩开二府；雍国青云亭覆灭中夺回遗楼并得仙术；接“金针案”出海擒武一愈，自创“八音焚海”，入囚海狱探监竹碧琼；天涯台海祭翻案、被危寻判入迷界洗罪，反杀碧珠婆婆并在迷界连斩统帅级海族；迷界突围后开第三内府得不周风，天涯台熬杀季少卿；获太虚使者身份并推动太虚角楼落地天府城；鸿蒙空间悟成声闻仙态并刻印第三内府；太庙“大师之礼”上击败雷占乾、阻截崔杼刺君；护国殿斩杀冒名张咏并卷入北衙问话；国库术库选得干阳之瞳，点将台受易星辰与重玄褚良指点完成火界；太虚幻境登顶最强内府并贡献火界；星月原遭平等国星楼蛊惑，观衍以玉衡星力护道；观河台以同字笺与叶青雨通信；点名林正仁为内府正赛首轮对手；内府正赛连胜项北、秦至臻与黄舍利夺魁展旗；凯旋太庙大典受如意仙衣封赏；奉旨锁拿柳啸并与田安平对峙；入长乐宫与太子论官道
- 关系网络：枫林五侠结义（凌河、杜野虎、赵汝成、方鹏举、向前）；亲属姜安安；盟友重玄胜、叶青雨；与姜无忧达成互助承诺；师徒决裂董阿；被杜如晦追凶；姜魇实为庄承乾残魂
- 现状：内府魁名已得，齐国凯旋封赏后入长乐宫与太子论官道（至第1200章）

# 叶青雨 (Ye Qingyu)

- 叙事重要度：重要配角
- 评分等级：主角
- 初登场：庄国玉衡峰篇
- 身份/出身：云国凌霄阁小姐，叶凌霄之女
- 所属/阵营：云国凌霄阁
- 修行/能力：名门修士，迟云山得神通“云篆”
- 关键经历：在玉衡峰任务中被姜望所救，赠“云中令”以答谢；枫林城域覆灭后协助姜望安置姜安安；迟云山争夺中与姜望并肩；姜望失踪期间照看安安并赠出云霄阁；受姜望所托前往九江玄甲欲接走杜野虎未果；随叶凌霄乘鹤辇赴观河台观礼；以“同字笺”与姜望通信，提醒杜如晦与林正仁风险
- 与主角关系：救命之恩与互相扶持的关系，亦是安安的重要依靠
- 现状：仍随叶凌霄观礼观河台，安安留在凌霄秘境修行（至第1150章）

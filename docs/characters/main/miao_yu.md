# 妙玉 (Miao Yu)

- 叙事重要度：主角
- 初登场：庄国枫林城篇（早期）
- 身份/出身：白骨道重要人物；三分香气楼人物（以“昧月”身份活动）
- 别称/化名：白莲（玉衡峰黑纱女子）；昧月（三分香气楼身份）
- 所属/阵营：白骨道
- 修行/能力：修为与能力深不可测，擅长隐秘行动与布局
- 关键经历：参与白骨道在枫林城的谋划；以“白莲”身份引姜望入玉衡峰；枫林城域覆灭前后与姜望有接触并令其带安安离开；以“昧月”身份在不赎城与凰今默接触、推动三分香气楼西境布局；升为天香后坐镇齐国天府城分楼，与香铃儿推进东域布局
- 与主角关系：敌对阵营的重要人物，但曾以“白莲”身份短期合作/引导
- 现状：以“昧月”身份在齐国天府城活动，寻求打通齐国上层关系（至第1050章）

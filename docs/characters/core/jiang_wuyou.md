# 姜无忧 (Jiang Wuyou)

- 叙事重要度：核心配角
- 评分等级：核心配角
- 初登场：齐国帝室线
- 身份/出身：齐国三皇女，华英宫主
- 所属/阵营：齐国帝室
- 修行/能力：武道修为强悍，擅使方天画戟，行事果决
- 关键经历：在华英宫练戟示人；答应全力协助姜望处理钓海楼海祭；近海群岛现身小月牙岛训诫晏抚并表态亲赴海祭；天涯台问座助姜望造势，出手提供“指舆”助其入迷界；太庙“大师之礼”中力挺姜望并回应齐帝问政；北衙问话时到场旁听护持；明确走“引武入道/真武之体”路线；太庙大典上代表帝室为姜望披授如意仙衣
- 与主角关系：合作互利，建立“一事换一事”的承诺
- 现状：帝室中支持姜望的重要力量之一，武道路线逐步成型（至第1188章）

# 《赤心巡天》世界观总览

本文件根据1-1200章信息整理，概括世界结构、政治格局、族群关系与主要冲突线，便于快速建立整体认知。

## 1. 宇宙结构与位面
- 现世：人族主世界，山河纵横、列国并起，长河为天下第一水脉。
- 幽冥：阴世之域，白骨道与幽冥神祇相关力量可渗透现世。
- 万界荒墓：魔祖陨落后的魔之墓园，上古魔窟通道可短暂连通。
- 太虚幻境：以太阴星力牵引的“星河世界”，拥有福地、演道台、论剑台等机制，消耗与产出以“功”为核心；太虚派为首倡与监察，后续开放太虚角楼扩张入口。
- 鸿蒙空间：太虚幻境更开放的交流空间，通过福地之门进入，当前仅小范围开放。
- 七星楼秘境：北斗七星对应七个世界，本卷涉及森海源界、隐元星沙海与天枢星浮陆。
- 云顶仙宫遗迹：近古九大仙宫之一，现为残破废墟，其复苏关键指向现世宗门遗留。
- 万妖之门：第二代人皇有熊氏与三位道尊所构筑的隔绝之门，天京城立其上以镇妖族。
- 沧海与迷界：现世唯一大海域为沧海，人族与海族主战场在迷界，外楼以下难以进入。
- 浮陆幽天：浮陆之下为幽天，星兽出没；天外来者可降临参与生死棋。

## 2. 地理与战略节点
- 长河与黄河之会：长河横贯现世，黄河之会是天骄成名舞台之一。
- 观河台：先贤镇压长河水脉之所，黄河之会举办地与资源分配枢纽。
- 六合之柱与“台上城”：观河台由六合之柱撑起天地，演武台分列其间，具“台上城”格局。
- 长河九镇传说：观河台附近流传“九镇”旧说，第五镇为狻猊桥。
- 临淄与稷下学宫：齐国帝都与学宫中枢，九卒轮戍都城。
- 太庙与护国殿：齐国皇室祭祀与国典核心场所，“大师之礼”在此举行。
- 阳国与青羊镇：乌祸扩散与齐阳战争的重要舞台。
- 云国与抱雪山：凌霄阁立宗于云城上空，云国以宗门为国本。
- 迟云山：三十三年一开，山内外时间流速差异明显，牵动神通果与仙宫遗留。
- 祁昌山脉与锁龙关：庄雍边境要冲，锁龙关为雍国南境雄关。
- 清江与澜河水系：水府势力盘踞，水陆两线可牵动国战大局。
- 上古魔窟：清江水底封存之地，可借道幽冥与万界荒墓。
- 大泽郡即城与七星谷：田氏祖地与七星楼秘境入口。
- 顺安府与青云亭：雍国西部顺安府为青云亭驻地，后期成为人魔屠门的关键节点。
- 和国与天马原：和国信奉原天神，风物温吞，三分香气楼亦有分楼。
- 星月原：现世接近星河之地，可积蓄星力。
- 丰城与齐馆/牧园/荆楼：沃国都城成为列国汇流枢纽，各强国设馆歇脚。
- 城域与官道：官道阵纹维系安全，城镇与资源地带沿官道布局。
- 海门岛：近海群岛最大中立岛，人族出海门户与商贸枢纽。
- 有夏岛/小月牙岛/得樵岛：近海群岛常见岛屿，五仙门、怒鲸帮等势力活动。
- 弦月岛与天涯台：近海群岛核心岛屿，海祭主祭地在天涯台。
- 丁未浮岛：迷界丁未区域唯一人族浮岛，由旸谷镇守，是迷界补给与避“异化”的关键节点。
- 浮图净土：重玄浮图以身重构规则所遗的人族共有净土，迷界中规则近似现世，各方势力设据点。

## 3. 政治格局与国家体系
- 列国格局：齐为强国，庄国体量较小，阳国为齐之属国，楚、秦等大国强者亦常现世；庄雍国战改变西境版图，庄国新设永昌郡。
- 齐国军政：九卒军制（秋杀军等）镇压四方，稷下学宫掌学术与话语权。
- 齐国中枢政务：政事堂九位朝议大夫议政，兵事堂节制九卒；北衙等衙门直受皇命。
- 皇朝礼制与“大师之礼”：太庙祭祀与御前武较合一，遴选黄河之会名额；黄河之会分内府、外楼、三十岁以下无限制三场。
- 六帝镇河与正赛主持：六大帝君以法相降临观河台，共同镇压长河；黄河之会正赛由玉京山真君余徙主持。
- 展旗与魁名荣誉：内府魁名可于观河台展皇朝旗帜，象征国运与资源分配话语权。
- 朱禾之盟：齐国神临级青牌涉案时可不经报备跨境追缉，后推及东域诸国。
- 名门与商盟：重玄、田氏、石门李、廉氏等世家掌资源与声望；四海商盟、聚宝商会左右商贸与政治。
- 雍国政局：韩殷战死后，韩煦掌权并定墨学为国学，墨门深度介入雍国体制。
- 雍国军政新格局：威宁候焦武与武功侯薛明义权势消长，雍庭对藩镇与边军敲打并行。
- 道属国体系：道门圣地与强国互相支撑，国家与宗门权力深度绑定。
- 官品与修行对照：修行境界与官品对应，封爵与功勋联动。
- 道院体系：城道院—郡道院—国道院构成官方修行教育与人才晋升体系。
- 镇海盟：钓海楼主危寻牵头成立，统合近海群岛势力的新型联盟，改变海上权力格局。

## 4. 修行与社会秩序
- 修者是国家武力与治理核心，道术、阵法、灵宝广泛影响民生与战事。
- 开脉丹、道勋、太虚“功”等资源，决定个人与国家成长速度。
- 太虚幻境对现世形成隐性影响：跨国切磋、功法推演与天骄竞争提前发生。
- 官道与官职体系为修行提供人气与资源支撑，外楼立楼可借官道人气锚定现世。

## 5. 宗门与学派
- 道门为主流，玉京山、大罗山、蓬莱岛等道统影响深远，道尊信仰根深。
- 儒门以书院传道，青崖书院为四大书院之一；法家圣地三刑宫以律令闻名。
- 龙门书院与暮鼓书院：四大书院中重要成员，龙门书院近观河台；暮鼓书院坐落书山脚下。
- 书山：儒门圣地，重学术清修，不广收门徒、不涉天下大势。
- 佛门以悬空寺为东圣地；墨门机关术与器物体系发达。
- 兵家以气血与战阵见长，与道修并为显流。
- 太虚派：隐世宗门，太虚幻境首倡与监察者，强调中立与超然。
- 凌霄阁：云国顶级宗门，阁主叶凌霄坐镇，牵动迟云山与仙宫遗留。
- 青云亭：雍国顺安府宗门，云顶仙宫遗楼所在，后被人魔屠门覆灭。
- 水府势力：清江水府为西境水系核心之一，水族力量可左右国战水线。
- 钓海楼：近海群岛霸主级宗门，主导海祭与海域秩序。
- 五仙门：有夏岛宗门，以“五感五殿”立宗，受钓海楼长老影响。

## 6. 非人族与异类
- 妖兽与凶兽：荒野主要威胁与资源体系，凶兽难猎且收益有限。
- 幽冥神祇与邪教：以白骨道为代表，图谋降世，危及现世秩序。
- 森海圣族：森海源界原住民，信奉龙神，存在“相狩”传统。
- 浮陆星兽：幽天涌出的灾厄之兽，为浮陆诸部族最大威胁。
- 魔与阴魔：上古魔潮遗留之敌，本应封绝边荒，现世出现水底魔窟异象。
- 真魔与血傀真魔：由魔窟与幽冥通道牵动的高阶魔类，具现世降临风险。
- 海族：中古时代龙族与水族分裂后的族群，退入沧海，长年与人族在迷界战场交锋。
- 海族阶层与海主本相：海族以战卒/战将/统帅/王爵/皇主为阶，海主本相为其核心天赋；近期演进至神魂层面，引发护宗海兽失控与战力跃升。
- 长河龙君：中古人皇敕封的水族共主，名义统御现世水族，权柄逐步式微，仅守长河水脉。
- 人魔：以九大人魔为代表的凶名修士群体，行为极端、手段残酷。
- 海兽：外海驯养的巨兽常用于护岛与威慑，失控则易引发灾难。

## 7. 资源与关键制度
- 开脉丹体系：修行入门关键资源，强国把控流通与分配。
- 道勋榜：道院任务与贡献体系的核心衡量工具。
- 福地与“功”：太虚幻境资源体系，影响功法推演与实力提升。
- 秘境体系：天府秘境预定神通种子，七星楼秘境决定增寿与外楼之秘归属。
- 神通果与仙宫遗留：迟云山神通果为稀缺机缘，灵空殿等遗留成仙宫复苏关键。
- 皇朝秘术与天子内库：以国之礼选拔天骄后可获赐，用于备战观河台。
- 九龙捧日永镇山河玺：六帝以此镇压长河水患，约十年一度再镇。
- 同字笺：可于百里范围内无道元波动通信的器物，用于修士隐蔽传讯。
- 演法阁：楚国名门可布设的大型推演法阵，允许在“幻境”中模拟术法与战斗。
- 龙币与龙骨船：近海群岛通行货币与航运工具由钓海楼主导，体现海上秩序。
- 迷晶与海疆榜：迷界基础资源“迷晶”支撑浮岛/海巢运转；海疆榜记录猎杀海族战绩，作为迷界功勋与洗罪指标。
- 海勋榜：海疆榜衍生的功勋榜单，正榜对应神临层次、副榜对应外楼层次，按战卒/战将/统帅/王爵计分以激励参战。
- 卫海士体系：镇海盟基于海勋设立九阶卫海士身份，形成新的海上功勋阶层。
- 太虚角楼：太虚幻境扩张入口，可让更多修士进入星河世界修行与切磋。

## 8. 主要冲突线
- 白骨道复燃与幽冥侵蚀：白骨尊神欲降世，庄国覆灭为导火索。
- 齐阳战争与乌祸：阳国矿脉与乌祸异变引爆齐阳大战。
- 齐国内部权斗：重玄胜与重玄遵之争，王夷吾与文连牧布局交锋。
- 七星楼秘境竞争：天罡地煞星位与秘境收获争夺。
- 云顶仙宫复苏线：仙宫遗留与迟云山传承牵动各方角力。
- 庄雍国战线：边境矛盾升级为国战，锁龙关成为胜负核心。
- 枫林旧案与师徒因果：姜望与董阿决裂，旧案余波持续牵动庄国权力结构。
- 清江水府魔窟线：水下阴魔与水府势力的潜在冲突。
- 无生劫对弈线：庄承乾与白骨尊神围绕无生劫的生死博弈。
- 列国征伐与资源竞争：强国压制小国，黄河之会仍为天骄舞台。
- 青云亭覆灭与人魔线：九大人魔为平衡之血屠灭青云亭，引发云顶仙宫新变。
- 沧海/海族战线：迷界常年战事与海祭传统，牵动近海群岛与齐国布局。
- 海族跃升与镇海盟线：海主本相神魂化推动海族跃升，钓海楼借势成立镇海盟统合海民。
- 近海宗门摩擦线：五仙门与怒鲸帮冲突不断，折射钓海楼在海上秩序的控制与缝隙。
- 天涯台生死决斗线：竹碧琼之死引发姜望与季少卿生死战，牵动齐国与钓海楼的规矩对抗。
- 太虚角楼扩张线：太虚派与齐国协作建设太虚角楼，带来修行资源与利益新格局。
- 田家暗线：乌列追查田焕文与田氏旧案，暗线牵动齐国内部秩序。
- 刺君与哭祠线：崔杼御前刺君、九返侯灵祠血书引爆舆论冲击，反齐势力暗线显露。
- 平等国暗线：刺君与哭祠背后组织浮出，并尝试以星光圣楼投射影响姜望道心。
- 黄河之会资源线：观河台争魁实为万妖之门后地盘与资源再分配。
- 景国弃赛与太虞疑云：景国内府天骄战死万妖之门后引发弃赛，各国对景国底牌与神秘天骄“太虞”猜测升温。
- 田柳旧怨与即城异变：柳啸袭即城、田安平内府炼城，田柳两家旧案再度激化。
- 官道与继承线：太子论官道、人气与功业，牵动皇权继承与修行路径选择。

## 9. 时间线概览（1-1200章）
- 两百年前：庄国高祖清洗白骨道，邪教隐而不绝。
- 现世前期：姜望在庄国崛起，白骨道之乱引发庄国覆灭。
- 现世中期：姜望入齐，卷入阳国矿脉与乌祸，齐阳战争爆发并终结。
- 现世后期：临淄权局与商盟角力升温，七星楼开启，秘境多世界展开。
- 现世补充：迟云山开启与云顶仙宫遗留现世；庄雍国战爆发并停战划界，雍国改立墨学；清江水府魔窟爆发、宋横江身死；云霄阁归位，云顶仙宫复苏线推进。
- 现世新进：姜望入雍国顺安府夺回青云亭遗楼，青云亭被九大人魔屠门；人魔“平衡之血”与卦师线索浮出；姜望经和国返临淄，海族/迷界与钓海楼海祭线全面展开。
- 现世近海：姜望接“金针案”出海，林有邪念尘追踪武一愈，尹观远程咒杀；有夏岛海兽失控与五仙门/怒鲸帮矛盾显化；小月牙岛“难说大师”骗局被揭，钓海楼大师兄陈治涛出面善后；囚海狱探监竹碧琼，海祭筹备推进。
- 现世海祭与迷界：天涯台海祭翻案、危寻现身并推动镇海盟成立；姜望与碧珠婆婆被送入迷界丁未洗罪，姜望斩杀统帅级海族引发白象王围岛，丁未区域战端将起。
- 现世迷界突围与天涯台决斗：姜望于界河斩杀鱼嗣庆与水鹰嵘完成洗罪，迷界位移至浮图净土后返回近海；竹碧琼油尽灯枯，姜望开第三内府得不周风并送其葬入天府秘境；海勋榜出世，姜望与季少卿天涯台生死决斗升级至两大强者隔空关注。
- 现世天涯台余波与角楼启幕：五仙门被屠引出田家暗线，竹碧琼自天府秘境复活并回钓海楼拜师；太虚派授太虚使者与角楼图纸，齐境公开太虚角楼落地天府城，姜望冲击太虚排名并谋求黄河之会名额。
- 现世太庙礼祭与暗线浮出：太庙“大师之礼”遴选黄河之会名额，重玄遵破境外楼；崔杼刺君与张咏哭祠事件牵出反齐暗线，北衙奉旨彻查。
- 现世黄河前夕：点将台特训、太虚最强内府更迭、星月原平等国星楼蛊惑被观衍破局；齐队出征至丰城齐馆，黄河之会资源与万妖之门地盘分配被明确。
- 现世观河台阶段：内府选拔赛完成正赛名单；景国内府天骄战死万妖之门后弃赛；六帝法相降临以九龙捧日永镇山河玺镇河；外楼正赛激战至无魁；内府正赛开幕，林正仁弃赛引发道属体系颜面危机。
- 现世观河台决赛与归朝：姜望夺内府魁名并展旗；太虞真人李一现身夺无限制场魁名；齐国凯旋太庙大典封赏；柳啸袭即城引发田安平内府炼城异变；太子论官道人气与外楼锚定。
